Image Requirements for iWare Soft Website

The following images are needed to complete the website. Place all images in the "images" folder.

1. Slider Images

You need 3 slider images:

  • slider-1.jpg - Professional image showcasing software development
  • slider-2.jpg - Mobile app development image
  • slider-3.jpg - Enterprise solutions/team collaboration image

Recommended size: 1920x1080px for each image

2. Hero Background Image

Filename: hero-bg.jpg

Description: A high-quality image related to software development, technology, or business professionals in a modern office environment.

Recommended size: 1920x1080px

3. About Section Image

Filename: about-img.jpg

Description: An image showing a team of professionals working together or your company office.

Recommended size: 800x600px

4. Portfolio Images

You need 4 portfolio images:

  • portfolio-1.jpg - ERP System project screenshot
  • portfolio-2.jpg - Healthcare Management Platform screenshot
  • portfolio-3.jpg - E-commerce Mobile App screenshot
  • portfolio-4.jpg - Financial Analytics Dashboard screenshot

Recommended size: 600x400px for each image

5. Company Logo (Optional)

Filename: logo.png

Description: Your company logo with transparent background.

Recommended size: 200x60px

Note: If you add a logo, you'll need to update the HTML to use it instead of the text logo.

How to Add These Images

1. Find or create the required images with the recommended sizes

2. Save them with the specified filenames in the "images" folder

3. The website is already configured to use these images automatically
